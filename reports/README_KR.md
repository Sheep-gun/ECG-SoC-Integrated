# Reports

- `INTEGRATED_TECHNICAL_REPORT_KR.md`: 제출 PDF 3~17쪽의 기술 본문을 문장과 표현 그대로 옮긴 공개 원고. PDF 밖의 보충 기술은 문서 마지막의 별도 절에만 수록한다.
- `INTEGRATED_TECHNICAL_REPORT_EVIDENCE_MAP.csv`: 문장, claim과 artifact 연결
- `INTEGRATED_REPORT_REVISION_AUDIT.md`: 최신 개정 범위
- `INTEGRATED_TECHNICAL_REPORT_REVIEW_CHECKLIST.md`: claim boundary 검토
- `PUBLICATION_READINESS_PREFLIGHT.md`: 공개 전 점검

대회 제출 PDF 원본은 개인정보와 서명을 포함하므로 저장소에 넣지 않는다. 저장소 문서의 기준 본문은 PDF의 기술 내용이며, 다른 `docs/` 문서는 이를 대체하는 재서술본이 아니라 근거와 재현성을 위한 보충 기록이다.
